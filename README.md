# dots 
requires oh my zsh

## how to install
clone to ~/dots
```
mv ~/.zshrc{,/bak}
echo '. ~/dots/zshrc' >> ~/.zshrc

```

## what are these folders?

### ./bin
file name = a command for your shell
```
extract = extract most archives
pbcopy = stdin -> clipboard
pbpaste = clipboard -> stdout
srm = safe rm (recycle bin)
far = find and replace
```
